1. Mensaje de bienvenida:
"¡Hola! Soy el asistente de SENATI. Te ayudaré a completar la encuesta sobre medios de transporte. Solo te tomará unos minutos."
2. Captura de datos (Capture blocks):
Bloque 1 - Nombre:

Speak: "¿Cuál es tu nombre?"
Capture: Variable {nombre} (texto)
Reprompt: "Por favor, dime tu nombre completo"

Bloque 2 - Apellido:

Speak: "¿Cuál es tu apellido?"
Capture: Variable {apellido} (texto)
Reprompt: "Por favor, dime tu apellido"

Bloque 3 - Escuela:

Speak: "¿En qué escuela o institución estudias?"
Capture: Variable {escuela} (texto)
Reprompt: "Por favor, indica tu institución educativa"

Bloque 4 - Transporte (Elección):

Speak: "¿Cuál es tu medio de transporte principal? Puedes elegir entre: Bus, Bicicleta, Auto, o Caminando"
Choice Block:

Bus → Set {transporte} = "Bus"
Bicicleta → Set {transporte} = "Bicicleta"
Auto → Set {transporte} = "Auto"
Caminando → Set {transporte} = "Caminando"


Else: Capture manual y set {transporte}

Bloque 5 - Comentario (Opcional):

Speak: "¿Tienes algún comentario adicional sobre tu medio de transporte? Si no, puedes decir 'ninguno'"
Capture: Variable {comentario} (texto)
No Reply: Set {comentario} = ""

3. Envío de datos (API Block):
Configuración del API Block:

Method: POST
URL: https://tu-app-render.onrender.com/api/voiceflow-submit
Headers:

  Content-Type: application/json

Body:

json  {
    "nombre": "{nombre}",
    "apellido": "{apellido}",
    "escuela": "{escuela}",
    "transporte": "{transporte}",
    "comentario": "{comentario}"
  }
Manejo de respuesta:

Success (200): "¡Perfecto! Tu respuesta ha sido guardada. Gracias por participar en nuestra encuesta de SENATI."
Fail (400/500): "Lo siento, hubo un problema al guardar tu respuesta. Por favor intenta nuevamente."

4. Mensaje final:
"¡Gracias por completar la encuesta! Tu opinión es muy importante para nosotros. ¡Que tengas un buen día!"
